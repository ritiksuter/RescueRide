# Mechanic Service

The Mechanic Service provides functionalities related to vehicle maintenance and repair management within the SOS Vehicle application.

## Structure

- **src/**: Contains the main application code.
- **controllers/**: Business logic for handling mechanic-related requests.
- **repositories/**: Data access layer for mechanic data.
- **routes/**: API routes for mechanic functionalities.
- **tests/**: Unit tests for the mechanic service.

## Docker

This service can be containerized using the provided Dockerfile.
