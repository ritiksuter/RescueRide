# User Service

The User Service manages user profiles and related functionalities within the SOS Vehicle application. It handles user data and interactions.

## Structure

- **src/**: Contains the main application code.
- **controllers/**: Business logic for handling user-related requests.
- **repositories/**: Data access layer for user data.
- **routes/**: API routes for user functionalities.
- **tests/**: Unit tests for the user service.

## Docker

This service can be containerized using the provided Dockerfile.
