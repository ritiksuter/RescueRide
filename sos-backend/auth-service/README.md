# Auth Service

The Auth Service is responsible for user authentication and authorization within the SOS Vehicle application. It manages user sessions and provides secure access to other services.

## Structure

- **src/**: Contains the main application code.
- **controllers/**: Handles authentication logic.
- **repositories/**: Data access layer for user data.
- **routes/**: API routes for authentication functionalities.
- **tests/**: Unit tests for the auth service.

## Docker

This service can be containerized using the provided Dockerfile.
