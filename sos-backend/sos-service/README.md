# SOS Service

The SOS Service provides emergency assistance functionalities for users of the SOS Vehicle application. It allows users to request help in case of emergencies.

## Structure

- **src/**: Contains the main application code.
- **controllers/**: Business logic for handling SOS requests.
- **repositories/**: Data access layer for SOS-related data.
- **routes/**: API routes for SOS functionalities.
- **tests/**: Unit tests for the SOS service.

## Docker

This service can be containerized using the provided Dockerfile.
