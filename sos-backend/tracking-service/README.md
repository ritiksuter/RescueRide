# Tracking Service

The Tracking Service provides functionalities for tracking vehicle locations and movements within the SOS Vehicle application.

## Structure

- **src/**: Contains the main application code.
- **controllers/**: Business logic for handling tracking requests.
- **repositories/**: Data access layer for tracking data.
- **routes/**: API routes for tracking functionalities.
- **tests/**: Unit tests for the tracking service.

## Docker

This service can be containerized using the provided Dockerfile.
