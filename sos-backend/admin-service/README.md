# Admin Service

This service handles administrative functionalities for the SOS Vehicle application. It includes features such as user management, statistics, and other admin-related tasks.

## Structure

- **src/**: Contains the main application code.
- **config/**: Configuration files for database and environment settings.
- **controllers/**: Business logic for handling requests.
- **repositories/**: Data access layer for interacting with the database.
- **routes/**: API routes for the admin functionalities.
- **services/**: Contains service logic for admin operations.
- **tests/**: Unit tests for the admin service.

## Docker

This service can be containerized using the provided Dockerfile.
