# Docker

This directory contains Dockerfiles for various services in the SOS Vehicle application. Each Dockerfile is used to build a container image for the respective service.

## Files

- **admin.Dockerfile**: Dockerfile for the Admin Service.
- **api.gateway.Dockerfile**: Dockerfile for the API Gateway.
- **auth.Dockerfile**: Dockerfile for the Auth Service.
- **mechanic.Dockerfile**: Dockerfile for the Mechanic Service.
- **notification.Dockerfile**: Dockerfile for the Notification Service.
- **sos.Dockerfile**: Dockerfile for the SOS Service.
- **tracking.Dockerfile**: Dockerfile for the Tracking Service.
- **user.Dockerfile**: Dockerfile for the User Service.
- **docker-compose.yml**: Docker Compose file for orchestrating multiple services.
