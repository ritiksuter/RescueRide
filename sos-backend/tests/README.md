# Tests

This directory contains unit tests for various services in the SOS Vehicle application. Each test file corresponds to a specific service and contains tests for its functionalities.

## Structure

- **admin.test.js**: Tests for the Admin Service.
- **auth.test.js**: Tests for the Auth Service.
- **mechanic.test.js**: Tests for the Mechanic Service.
- **sos.test.js**: Tests for the SOS Service.
- **tracking.test.js**: Tests for the Tracking Service.
- **user.test.js**: Tests for the User Service.
